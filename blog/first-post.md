# My First Blog Post

This is an example blog post written in Markdown.

## What is this?

This is a minimalist blog setup that allows you to:

1. Write blog posts in Markdown
2. Simply commit them to the repository
3. They automatically appear on your blog

## How to add a new post

To add a new blog post:

1. Create a new `.md` file in the `blog/` directory
2. Edit `blog/index.html` and add your post to the `posts` array at the top of the script
3. Commit and push - that's it!
